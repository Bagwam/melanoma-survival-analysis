# Melanoma Survival Statistical Analysis

## Project Overview

This project explores statistical relationships between clinical variables in a melanoma dataset using R.

The analysis investigates whether factors such as tumor thickness, age, and gender influence survival outcomes.

## Dataset

The dataset contains information on melanoma patients including:

- Survival time
- Age
- Sex
- Tumor thickness
- Ulceration status

## Methods Used

The following statistical methods were applied:

- Exploratory Data Analysis
- Pearson correlation analysis
- Linear regression modelling
- Welch two-sample t-tests
- Normality testing using QQ-plots

## Tools

- R
- ggplot2
- dplyr

## Key Findings

- Tumor thickness shows a weak negative correlation with survival time.
- Female patients exhibited longer survival times than males.
- Tumor thickness distribution is positively skewed.
